# TradeSphere

**TradeSphere** is a premium, AI-powered financial research and trade management platform. It was developed as a flagship project for **AJ Consultancy** to modernize trading operations and market analysis.

## 📄 Documentation

A complete, single-page documentation covering the technology stack, core features, architecture, and setup instructions can be found here:

👉 **[View Complete Documentation](./DOCUMENTATION.md)**

## 🚀 Quick Start

1. **Backend**:
   - `npm install`
   - Configure `.env`
   - `npm run dev`
2. **Frontend**:
   - `cd client`
   - `npm install`
   - `npm run dev`

## ✨ Features at a Glance
- AI Stock Analysis (Gemini 1.5 Flash)
- Live Data Scraping (Screener.in)
- Professional Trade Journaling
- Client Capital Management
- Live Market Charts (TradingView)
- Financial News Aggregator
